Дефекты в классе StringUtils:

1. Метод trim():
   - Ожидание: Удаляет все пробелы в начале строки
   - Реальность: Удаляет только один пробел
   - Пример: trim("   skypro") возвращает "  skypro" вместо "skypro"

2. Метод contains():
   - Ожидание: Регистронезависимый поиск
   - Реальность: Регистрозависимый поиск
   - Пример: contains("SkyPro", "s") возвращает False вместо True

3. Метод delete_symbol():
   - Ожидание: Удаляет все вхождения символа
   - Реальность: Удаляет только первое вхождение
   - Пример: delete_symbol("hello", "l") возвращает "helo" вместо "heo"

4. Метод is_empty():
   - Ожидание: Возвращает True для строк из пробелов
   - Реальность: Возвращает False для непустых строк с пробелами
   - Пример: is_empty("   ") возвращает False вместо True

5. Метод list_to_string():
   - Ожидание: Корректно обрабатывает пустой список
   - Реальность: Вызывает исключение при пустом списке
   - Пример: list_to_string([]) вызывает TypeError
